Cache and return a FTX HTTP client with the given key or secret.

If a cached client with matching key and secret already exists, then that cached client will be returned.

  • clock ( LiveClock ) – The clock for the client.

  • logger ( Logger ) – The logger for the client.

  • subaccount ( str , optional ) – The subaccount name. If None then will source from the FTX_SUBACCOUNT env var.

  • us ( bool , default False ) – If the client is for FTX US.
